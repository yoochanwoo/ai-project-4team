자바와 파이썬의 차이

1. 실행방식
    자바
    - 컴파일 방식 : 전체 코드를 번역한 후 실행(하나라도 오류가 있으면 전체가 실행되지 않음)
    파이썬
    - 인터프리터 방식 : 코드를 한 줄씩 실행(오류가 있는 부분에서 실행이 멈춤)
2. 문법
    자바
    - 중괄호({})로 블럭구분
    - 세미콜론으로 문장 끝맺음
    - 규칙이 엄격함
    파이썬
    - 간단하고 영어문장과 유사
    - 들여쓰기와 콜론(:)으로 블럭 구분
3. 속도
    자바
    - 비교적 느림
    파이썬
    - 비교적 빠름
4. 데이터 타입
    자바
    - 정적타입 : 변수타입이 고정됨
    파이썬
    - 동적타입 : 타입을 자유롭게 변경할 수 있음
5. 메모리 관리
    자바, 파이썬
    - 자동(가비지컬렉션)
6. 용도
    자바
    - 기업용 소프트웨어
    - 대규모 시스템
    파이썬
    - 데이터 분석
    - 인공지능
    - 웹 개발
    - 자동화 스크립트
7. 코드 작성
    자바
    - 코드가 길고 많아 오래걸림
    파이썬
    - 비교적 간단하고 적어 빠름

파이썬의 문법

5. 리스트(list)
    - 순서가 있는 집합
    - 수정 가능
    - 중복 허용
    - 대괄호([])로 묶임
6. 튜플(tuple)
    - 순서가 있는 집합
    - 수정 불가
    - 중복 허용
    - 괄호(())로 묶임
7. 딕서너리(dict)
    - 키와 값의 쌍
    - 중괄호로 묶임
    - json과 비슷

블럭구분

자바
    - 중괄호({})로 구분
파이썬
    - 콜롬(:)과 왼쪽으로부터 떨어진 거리로 구분
